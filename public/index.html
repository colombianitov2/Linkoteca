Nube y sincronización

Para usar la unidad de red, elige IP / servidor propio y pega una ruta como P:\Linkoteca o \\100.75.146.94\HDD_PROGRAMAS\Linkoteca.

Papelera

Los enlaces eliminados se pueden restaurar desde Papelera antes de que se cumpla este plazo.

Instrucciones rápidas

  1. Pega un enlace, elige carpeta y pulsa Agregar.
  2. Haz clic en una tarjeta o en Editar para corregir título, descripción o carpeta.
  3. Usa Copiar para compartir el enlace original.
  4. Selecciona varios enlaces para eliminarlos o restaurarlos en grupo desde Papelera.
  5. En Android, usa Compartir desde otra app y elige Linkoteca para guardar el enlace.
  6. Configura Nextcloud o IP / servidor propio para sincronizar el banco de enlaces.

Conectar Linkoteca

Para usar la app desde celular, abre Linkoteca en el PC y escribe la IP del servidor local.
