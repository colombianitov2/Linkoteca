Instrucciones rápidas

  1. Pega un enlace, elige carpeta y pulsa Agregar.
  2. Haz clic en una tarjeta o en Editar para corregir título, descripción o carpeta.
  3. Usa Copiar para compartir el enlace original.
  4. Usa Exportar para respaldar e Importar para recuperar tus enlaces.
