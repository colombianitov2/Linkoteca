Instrucciones rápidas

  1. Pega un enlace, elige carpeta y pulsa Agregar.
  2. Usa Actualizar vistas para buscar miniaturas y descripciones.
  3. Haz clic en una tarjeta o en Editar para corregir título, descripción o carpeta.
  4. Usa Copiar para compartir el enlace original.
  5. En Android, usa Compartir desde otra app y elige Linkoteca para guardar el enlace en Por revisar.
  6. Configura Google Drive, Nextcloud o una carpeta local para sincronizar el banco de enlaces.

Conectar Linkoteca

Para usar la app desde celular, abre Linkoteca en el PC y escribe la IP del servidor local.
